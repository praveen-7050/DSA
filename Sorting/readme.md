sorting is the method that arranges the elements in a specific order 

There are two types of sorting 
1. In place sorting 
2. Out place sorting

In place sorting is the sorting method that does not require any extra space

out place sorting is the sorting method that requires extra space

in sorting there are many types of sorting

bubble sort
selection sort
insertion sort
merge sort
quick sort etc..

Bubble sort is the sorting method that compares the adjacent elements and swaps them if they are in the wrong order
